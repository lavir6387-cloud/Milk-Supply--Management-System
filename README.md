
# 🥛 Milk Supply Management System

A simple and user-friendly web application for managing daily milk supply records, customer details, billing, and reports.

## 🚀 Live Demo

https://lavir6387-cloud.github.io/Milk-Supply--Management-System/

---

## 📌 Features

### 👥 Customer Management
- Add new customers
- Store customer details
- View customer records

### 🥛 Milk Entry Management
- Record daily milk supply
- Track milk quantity
- Maintain supply records

### 💰 Bill Generation
- Generate customer bills automatically
- Calculate total amount
- Print bills
- Download bills

### 📊 Dashboard
- Overview of milk supply data
- Quick access to all modules
- User-friendly interface

### 📈 Reports
- View supply records
- Monitor customer transactions
- Track billing information
